# Cookie Craze

Projet React + Vite + Tailwind + Framer Motion.

## Lancer en local

```bash
npm install
npm run dev
```

puis ouvre l'URL affichée (par défaut http://localhost:5173).

## Build de prod

```bash
npm run build
npm run preview
```

## Déploiement

- **Vercel** : importez ce repo → Deploy.
- **Netlify** : `npm run build` puis déployez le dossier `dist/`.
